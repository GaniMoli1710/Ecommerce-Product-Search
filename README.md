# E-commerce Product Search Relevance (LLM + FAISS + RAG)

## Project Overview

This project uses:
- WANDS dataset (Wayfair)
- LLM for semantic embeddings
- FAISS for vector search
- Together API for query-based product summaries
- Streamlit for UI
