# E-commerce Product Search Relevance (LLM + FAISS + RAG)

## Project Overview

This project uses:
- WANDS dataset (Wayfair)
- MiniLM for semantic embeddings
- FAISS for vector search
- OpenAI API for query-based product summaries
- Streamlit for UI
